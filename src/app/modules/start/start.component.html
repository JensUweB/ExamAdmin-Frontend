Welcome to our exam admin!
What is this all about?
This is a progressive web app for administrating martial arts exams. You can:

  • Create martial arts
  • Create exams
  • Create user accounts
  • Users can register themselfs as participants for an existing exam
  • Examiners can upload exam results and upload a protocol for each user
